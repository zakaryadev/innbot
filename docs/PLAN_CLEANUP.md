# План очистки проекта (Cleanup Plan)

## Задача
Удалить все временные скрипты и исходные файлы данных, оставив только те файлы, которые нужны для работы самого бота в production.

## Файлы к удалению (To be deleted)
1. `test_fuzz.py` — тестовый скрипт для проверки алгоритма Левенштейна.
2. `import_csv_to_db.py` — одноразовый скрипт, которым мы залили CSV в базу.
3. `transliterate_csv.py` — одноразовый скрипт перевода кириллицы в латиницу.
4. `Нокис хамме мекемелер.csv` — исходный сырой файл данных.
5. `Нокис хамме мекемелер_latin.csv` — обработанный файл данных (данные уже внутри `data.db`).
6. `db_setup_excel.py` — старый скрипт для парсинга `.xlsx` файлов (эксель-файлов у нас больше нет, бот работает напрямую с готовой базой `data.db`).

## Файлы, которые останутся (To keep)
- **Основной код**: `bot.py`, `database.py`, `config.py`
- **База данных**: `data.db`
- **Инфраструктура**: `Dockerfile`, `docker-compose.yml`, `entrypoint.sh`, `.env`, `.env.example`, `requirements.txt`, `.gitignore`, `.dockerignore`, `README.md`.
